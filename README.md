`concat_video.ps1`

Usage
```
./concat_video.ps1 OUTPUT_FILENAME INPUT_FILENAME_1 INPUT_FILENAME_2 [INPUT_FILENAME_3 [INPUT_FILENAME_4 [...]]]
```

Example:
```
./concat_video.ps1 the_life_of_pi.mp4 the_life_of_pi-A.mp4 the_life_of_pi-B.mp4 the_life_of_pi-C.mp4
```
